Commander

Commander is a Discord slash command generator for JavaScript and Python. Make your bot's commands easy to use with slash commands, which integrate commands seamlessly into the Discord client. With slash commands, you can add documentation to your command in Discord. Forget having to worry about sanitization and type checking and let Discord handle it for you via Discord's slash command argument system. Arguments, like commands, are seamlessly integrated into the client and make specifying options a breeze.

Commander currently does not support Discord.js or Discord.py. Commander is also currently in development, so you wouldn't be able to generate commands yet anyway.
